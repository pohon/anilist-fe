
## How to run locally

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.


## Tech stacks

- Next JS
- React
- GraphQL
- Tailwind CSS
- Jest & React Testing Library
